## 1. Student Journey

### 1.1 Discovery → Employment

```
Awareness          Registration         Engagement           Outcome
    │                   │                    │                  │
    ▼                   ▼                    ▼                  ▼
Find Cyra Nexus → Create Account → Build Profile → Get Hired/Internship
(via hub/social)   (select Student)  (skills, portfolio)  (job offer)
                        │                    │
                        ▼                    ▼
                  Verify Email        Enroll in Hub Course
                        │                    │
                        ▼                    ▼
                  Complete Profile    Apply to Jobs/Internships
                        │                    │
                        ▼                    ▼
                  Explore Dashboard   Book Mentor Session
```

**Key Touchpoints:**
1. Landing page → Register CTA
2. Role selection → Student
3. Profile wizard (skills, education, portfolio)
4. Student dashboard (opportunities feed)
5. Job/internship application flow
6. Mentor booking calendar
7. Notification: application status update

**Pain Points to Address:**
- Overwhelming number of opportunities → AI matching & filters
- Incomplete profile blocks applications → progress indicator
- No feedback on applications → status tracking + notifications
